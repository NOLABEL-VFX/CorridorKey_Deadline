# CorridorKey DeadlinePlugin Files

This folder contains the Deadline plugin package for CorridorKey:

- `CorridorKey.options`: submission-time job options.
- `CorridorKey.param`: plugin configuration (executables, repository path, OCIO config).
- `CorridorKey.py`: Deadline plugin runtime implementation.

Install this folder under your Deadline Repository custom plugins path and configure defaults per worker environment.
